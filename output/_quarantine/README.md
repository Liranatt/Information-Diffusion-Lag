# Quarantined outputs

These CSVs have **no generating script** anywhere in the repo (`grep -rn "jan_mar" --include=*.py`
returns nothing). They cannot be regenerated or verified, so they must not be cited as evidence.

They predate the T4 experiments and the weekend-entry calendar fix, so any figure in them was
computed on a candidate universe of 1121 rows (the current one is 1192) with ~3,996 weekend
signals silently dropped.

Retained only as an artifact. Delete once nothing references them.
